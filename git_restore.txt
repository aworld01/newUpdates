git restore --source=<commit-hash> <file>: to go at that commit's a specific file point.

git restore --source <commit-hash> <file>: to go at that commit's a specific file point.

git restore -s <commit-hash> <file>: to go at that commit's a specific file point.

git restore -s <commit-hash> .: to go at that commit point.

git restore -s HEAD: to go at last commit point.

git restore -s HEAD~1: to go one commit back.

git restore -s HEAD~2: to go two commits back.


Syntax:
    git restore --source=a5c945f file1.txt: restore file1.txt
    
    git restore --source a5c945f file1.txt: restore file1.txt

    git restore -s a5c945f file1.txt: restore file1.txt

    git restore -s a5c945f .: restore all files

    git restore -s HEAD: back at last commit
    
    git restore -s HEAD~1: 1 commit back

    git restore -s HEAD~2: 2 commit back
